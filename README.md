## Description

dm-types-legacy is a collection of legacy data types for working with
legacy databases or unsanitized data.

## Requirements

* [dm-core](http://github.com/datamapper/dm-core/) ~> 1.0.0

## Install

    $ sudo gem install dm-types-legacy
